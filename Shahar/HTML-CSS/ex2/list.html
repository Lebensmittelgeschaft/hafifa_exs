This is the list page

There are two main kinds of lists:

  1. The first kind is the ordered list

    The ordered list is used for organizing objects where the order matters, or the counting is important.
    An exmple for this use would be the list of LOTR movies by their order:

    1. The Fellowship of the Ring
    2. The Two Towers
    3. The Return of the King

  2. The second kind is the unordered list

    The unordered list is used for organizing objects where the order doesn't matter,
    or if the #PC forbids you from favoring one item over another.
    An exmple for this use would be the list of Hogwarts houses, since there should not be one more important than the other.
    (though everyone knows Hufflepuff is the best):

    • Gryffindor
    • Hufflepuff
    • Ravenclaw
    • Slytherin
